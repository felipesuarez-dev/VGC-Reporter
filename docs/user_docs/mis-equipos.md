# Mis equipos

Listado de los equipos que has guardado.

## Qué verás

- Una card por equipo con su nombre, formato, los 6 sprites y la fecha de última edición.

## Qué puedes hacer

- **Editar** un equipo: te lleva al *Constructor de equipos* con el equipo cargado.
- **Borrar** un equipo (con confirmación).
- **Crear nuevo**: atajo al constructor vacío.

## Dónde se guardan los datos

Los equipos se guardan localmente en una base de datos SQLite dentro de los datos de la aplicación. **No se suben a ningún servidor**. Si reinstalas la aplicación o cambias de PC sin migrar el archivo, perderás los equipos guardados.

Para hacer una copia de seguridad: usa el botón de exportar en el Constructor para cada equipo y pega el texto Showdown en un lugar seguro (un archivo de texto, un repo privado, etc.).
